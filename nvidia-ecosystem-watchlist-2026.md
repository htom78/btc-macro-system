# NVIDIA 生态链筛选器：从果链逻辑到 AI Factory 链

口径：这是“器层”工具，不是买卖建议。近 1 年股价表现来自 2025-06-03 附近到 2026-06-02/03 的行情粗算，跨市场按本币股价口径看，不纳入汇率变动。

## 评分框架

| 指标 | 看什么 | 高分含义 |
|---|---|---|
| NVIDIA 订单联动 | 收入是否直接受 GPU/AI factory 出货拉动 | NVIDIA 出货增加，公司订单能直接增加 |
| 瓶颈程度 | 这个环节是不是供给卡点 | 扩产慢、认证难、替代少 |
| 议价权 | 能不能涨价或守住毛利率 | 不是只做低毛利代工 |
| 订单能见度 | backlog、长约、客户确认度 | 未来 4-8 个季度收入更容易看清 |
| 估值热度 | 股价是否已提前兑现太多 | 涨幅越大、估值越高，追高风险越高 |
| 替代/过剩风险 | 技术路线或扩产是否可能反转 | 一旦供需缓和，利润会不会掉得很快 |

简单使用方法：

```text
真机会 = 订单联动 + 瓶颈程度 + 议价权 + 订单能见度
真实风险 = 估值热度 + 替代风险 + 客户集中 + 扩产过剩
```

## 分层结论

### A 类：真正值得长期跟踪的瓶颈

HBM、TSMC/先进封装、AI 网络芯片、高速光互联、电源/液冷。

这类公司最接近 AI factory 的“物理瓶颈”。但其中很多已经涨幅巨大，接下来不能只看故事，要看订单、毛利率和扩产节奏。

### B 类：订单弹性强，但利润弹性未必强

服务器 OEM/ODM、部分封测、部分制造服务。

这类最容易出现“收入大涨、利润没有同等大涨”。重点看毛利率、存货、应收账款和经营现金流。

### C 类：第二层客户，不一定是 NVIDIA 生态的直接受益者

云厂商和 AI 云公司花钱买 NVIDIA，但未必马上赚钱。核心问题是：AI capex 能不能转成收入、利润和现金流，而不是只变成折旧。

## 必看死角

1. 客户集中：订单来自一个大客户时，股价弹性大，砍单风险也大。
2. 毛利率：不要把收入增长误读为利润增长。
3. 存货：AI 供应链一旦补库存过头，库存会先暴露。
4. 资本开支：扩产越猛，未来过剩风险越大。
5. 技术替代：HBM、光互联、铜互联、ASIC、液冷路线都可能改变利润分配。
6. 合同质量：长期订单不等于锁定利润，要看价格、取消条款和客户付款能力。
7. 估值热度：涨幅越大，后续越需要业绩持续超预期。

## 下一步研究顺序

1. 先选 8-10 个 A 类核心标的做财报拆解：SK Hynix、Micron、TSMC、Broadcom、Marvell、Coherent/Fabrinet、Vertiv、Delta。
2. 每家公司只看 6 个指标：AI 收入占比、毛利率变化、backlog、capex、存货、经营现金流。
3. 把“涨幅最大”与“基本面最硬”分开排名。
4. 最后再进入术层：只在估值、业绩和回撤位置同时合适时，才考虑交易。
