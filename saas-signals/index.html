How to update

怎么更新

  1. 财报出来后，跑更新脚本（自动写历史 + 更新看板）：
python3 saas-signals/scripts/\nupdate_signal.py \\\n  --ticker SNOW --signal grossmargin \\\n  --value "74.2%" --status amber \\\n  --asof 2026-08-26 \\\n  --source "Q2 FY27" --note "..."
  1. signal: growth / grossmargin / nrr　status: green / amber / red
  2. git push 即触发 Pages 自动部署。
  3. 看当前：--show；也可直接编辑 data/signals.json。
