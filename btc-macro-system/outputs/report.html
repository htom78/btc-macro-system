核心读法 / Core Read

当美元流动性扩张、实际利率下行、美元走弱且信用压力受控时，BTC 的宏观环境通常更友好。Persistent inflation can block easing, and high debt can shift from a long-term hard-money narrative into a short-term rate-pressure problem.

这个系统判断的是环境，不是给出买卖点。It reads the regime, not a trade entry.
