# Re-underwrite

每年一次,每个标的半小时,只回答 3 个问题。10-year OG 持有假设下的最低维护成本。

## 为什么

"没动过 ≠ 纪律"。不复核的 buy-and-hold 是 buy-and-forget,thesis 可能已经悄悄变了你没察觉。Bessembinder:1926 年以来 4% 公司贡献全部美股财富 — 但那 4% 里也会有公司**突然不再是 4%**(GE / IBM / Intel 都曾在里面)。信号会出现,只是没人盯着的话不会自动跳出来。

详见 memory: `bogle-indexing-position.md` + `chamath-investing-framework.md` 的 re-underwrite 条目。

## 节奏

- **每标的一年一次**,固定触发日(财报日 / 标的"生日" / 你自己挑的日子)
- **半小时上限**,超时停 — 想不清楚说明 thesis 本身糊
- 在对应 ticker 文件追加新 section(不要覆盖旧的 — 历史复核记录本身就是数据)

## 模板

每次复核必答 3 个问题(见 `_template.md`):

1. **当初买的核心 thesis 是什么?**(写不出说明从来没明确过)
2. **过去 12 个月有什么硬数据改变了这个 thesis?**(基本面 / 战略 / 管理层 / 监管 — 不是股价)
3. **如果今天第一次看到这家公司,以现价我还会买吗?**(不会 → 至少诚实回答为什么没卖)
