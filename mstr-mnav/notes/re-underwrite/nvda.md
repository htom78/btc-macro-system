## NVDA 特有的复核字段(Q2 补充)

NVDA 的 thesis 是"AI 训练 + 推理 infra 的定价权 winner"。每次复核必过:

- **客户集中度**:hyperscaler(MSFT / META / GOOG / AMZN)合计占营收百分比,以及他们各自的 capex guide
  - 任何一家 hyperscaler 给出 capex 下调或自研 ASIC 大规模上量信号 → thesis 削弱
- **竞争格局**:
  - AMD MI300 / MI400 系列市占进展
  - Google TPU / AWS Trainium / MSFT Maia 自研芯片进展
  - 中国本土 GPU(华为 Ascend / 寒武纪)在中国市场的替代速度(NVDA 中国营收占比变化)
- **毛利率走向**:数据中心业务毛利率 — 70%+ = 定价权完整,持续下行 = 竞争加剧信号
- **CUDA 护城河**:开发者迁移 ROCm / Triton / OpenAI Triton-equivalent 的速度。CUDA 锁定如果被打破,thesis 根基松动
- **监管 / 出口管制**:美国对华芯片限制升级 / 反垄断调查
- **AI 需求曲线本身**:模型 scaling 是否还在 working?推理 vs 训练 mix 变化(推理对 NVDA 的护城河相对更弱)
